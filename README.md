# Jessy Care Clinic (React + TypeScript + Tailwind + Framer Motion)

Starter clinic website scaffold with:

- React + TypeScript (Vite)
- Tailwind CSS
- Framer Motion animations
- Clinic-focused homepage sections

## Run locally

1. Install Node.js (includes `npm`) if not installed.
2. Install dependencies:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Starter sections included

- Hero + clinic highlights
- Core services grid
- Doctor team cards
- Appointment flow steps
- Final call-to-action section
